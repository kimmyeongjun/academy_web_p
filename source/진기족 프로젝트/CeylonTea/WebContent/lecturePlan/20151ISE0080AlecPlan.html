교과목
목표
(한글 1000자)

리눅스 시스템에서 운영되는 네트워크 서버를 구축하고, 실무에서 운영되는 환경과 동일하게 운영한다.

교과목 개요
(한글 1000자)

- Linux 기본 명령어를 배우기 보다는, 실무에서 사용되는 Linux 명령을 위주로 배운다.
- Linux 네트워크 서버를 모든 수강생이 직접 구현하게 됨.
- 여러대의 서버를 개인별로 직접 운영하게 됨.
- 실무 환경과 동일한 서버환경을 구축해서 학습함.
